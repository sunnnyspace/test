❓ 核心问题 | 핵심 질문 (9개)
1
基本信息
这是您第几次来韩国？
쩌 스 닌 디 지 츠 라이 한궈?
한국 방문이 몇 번째예요?
📌 예상 답변
第一次 (띠 이 츠) / 第二次 (띠 얼 츠) / 来过好几次了 (라이 궈 하오 지 츠 러)
2
翻译/번역최우선 ⭐
在韩国旅行时用翻译APP吗？用哪个？
짜이 한궈 뤼싱 스 용 판이 APP 마? 용 나 거?
여행 중 번역 앱 써요? 어떤 거?
📌 예상 답변
用Papago (용 파파고) / Google翻译 (구거 판이) / 百度翻译 (바이두 판이) / 有道 (요우다오) / 没怎么用 (메이 전머 용)
3
翻译/번역최우선 ⭐
您用过Papago吗？感觉怎么样？
닌 용 궈 파파고 마? 간쥐에 전머양?
Papago 써보셨어요? 어떠셨어요?
📌 예상 답변
用过，挺好用的 (용 궈, 팅 하오 용 더) / 没用过，一般用Google翻译 / 听说过，但还没用 (팅 슈어 궈, 단 하이 메이 용)
4
翻译/번역최우선 ⭐
翻译APP在哪些情况下最需要？
판이 APP 짜이 나 시에 칭쾅 샤 쭈이 쉬야오?
번역 앱 어떤 상황에서 가장 필요해요?
💬 후속 질문
比如点菜？购物？还是问路？
(비루 디엔 차이? 고우 우? 하이 스 원 루?)
📌 예상 답변
点菜的时候 (디엔 차이 더 스허우) / 购物的时候 (고우 우 더 스허우) / 看不懂菜单 (칸 부 동 차이 딴)
5
导航/내비게이션
用地图APP时遇到过困难吗？
용 띠투 APP 스 위 따오 궈 쿤난 마?
지도 앱 쓸 때 어려움 있었어요?
💬 후속 질문
比如找不到路？还是不知道用哪个APP？
(비루 자오 부 따오 루? 하이 스 부 즈다오 용 나 거 APP?)
📌 예상 답변
Naver地图不太会用 (네이버 띠투 부 타이 후이 용) / 地铁标识看不懂 (띠티에 비아오즈 칸 부 동)
6
支付/결제
购物时主要怎么付款？现金？卡？还是支付宝？
고우 우 스 주야오 전머 푸 관? 시엔 진? 카? 하이 스 즈푸바오?
쇼핑 시 주로 어떻게 결제해요? 현금? 카드? 알리페이?
📌 예상 답변
刷卡 (솨 카) / 现金 (시엔 진) / 支付宝可以用 (즈푸바오 커이 용) / 微信支付 (웨이신 즈푸)
7
支付/결제

미엔 쉐이 투이 쉐이 리우 청 순 리 마?
면세 환급 과정 순조로웠어요?
💬 후속 질문
如果遇到问题，是什么问题？
(루 궈 위 따오 원 티, 스 선머 원 티?)

还行，挺简单的 (하이 싱, 팅 지엔 단 더) / 不太懂流程 (부 타이 동 리우 청) / 不知道在哪里退税
8
信息来源/정보 출처
您在哪儿找的韩国旅游信息？
닌 짜이 나얼 자오 더 한궈 뤼요우 신시?
한국 여행 정보 어디서 찾으셨어요?
💬 후속 질문
小红书？YouTube？还是旅游网站？
(샤오홍슈? 유투브? 하이 스 뤼요우 왕잔?)
📌 예상 답변
小红书 (샤오홍슈) / 看YouTube (칸 유투브) / 朋友推荐 (펑요우 투이지엔) / 抖音 (도우인) / 马蜂窝 (마펑워)

开放式/개방형핵심 ⭐
在韩国旅行中，什么最不方便？
짜이 한궈 뤼싱 중, 선머 쭈이 부 팡비엔?
한국 여행 중 뭐가 가장 불편했어요?
